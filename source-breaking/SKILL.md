# Source Breaking

Use for breaking/current general news where freshness matters.

## Source pattern

Use live web/search + primary/official/wire/reputable sources.

Prioritize official statements, agencies, courts, governments, companies, regulators, and reputable/wire outlets when primary sources are not yet available.


## Reference links

Use current web/search first, then prefer primary sources. Useful starting points:

- AP News: https://apnews.com/
- Reuters: https://www.reuters.com/
- BBC: https://www.bbc.com/news
- U.S. government releases: https://www.usa.gov/agency-index
- SEC filings/search: https://www.sec.gov/edgar/search/

## Workflow

1. Search current web for the event/topic.
2. Open/fetch the best primary or reputable sources.
3. Record publication/update time and retrieval time.
4. Cross-check major claims with at least two credible sources, or mark as unverified.
5. Send normalized source notes to `source-briefing`; for chat reports, use its Telegram-readable pattern and do not call the output a visual asset/brief.

Avoid social-only claims, screenshots without source links, repost chains, and low-quality aggregators.
