# Source Breaking

Use for current events, policy/regulatory updates, crises, fast-moving company/market developments, and breaking news.

Read first:
- `../source-common/references/research-contract.md`
- `../source-common/references/stop-rules.md`
- `../source-common/references/source-quality-ladder.md`
- `../source-common/references/citation-and-timestamp-rules.md`
- `../source-common/references/safety-boundaries.md`

Workflow:
1. Identify event, geography/entity, and freshness window.
2. Prefer official/primary statements and reputable current reporting.
3. Preserve publish/update/retrieval times.
4. Cross-check major claims with two credible sources or mark uncertainty.
5. Send normalized source notes to `source-briefing`.

Output:
- event scope
- source notes with timestamps
- confirmed claims
- unverified/contradicted claims
- missing evidence
- citation/source link
