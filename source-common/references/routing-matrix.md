# Routing Matrix

Use this to choose the first source lane, not the full research story.

| Signal | First collector lane |
|---|---|
| broad topic/digest/RSS/source list | `source-topic` |
| current/breaking/news/policy/regulatory | `source-breaking` |
| AI/tech/startup/builder ecosystem | `source-tech` |
| papers/literature/research trend | `source-research` |
| repo/library/release/GitHub trend | `source-github` |
| product/launch/competitor/positioning | `source-product` |
| finance/BTC/ticker/funding/earnings | `source-market` |
| X/social/community sentiment or early lead | `source-social-signal` |
| curated recurring sources | `source-watchlist` |

Optional end steps:

- `source-briefing`: final synthesis from gathered notes.
- `source-visual`: visual evidence, charts, screenshots, PDF figures, or requested visual output.
- `source-artifact`: save/report/handoff/archive after briefing.

Tie-breaks:

- primary/current source before social signal
- official product/company/repo docs before commentary
- source-gathering before briefing
- briefing before artifact/visual-prompt output
- social is signal only, never sole proof
