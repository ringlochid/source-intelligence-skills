# Visual Evidence Rules

Use visual reads only when image/layout/chart/table/figure evidence matters.

## Rules

- source collection and dedupe come first
- inspect only selected important or unclear sources
- send or attach image(s) first when producing a visual read
- caption in one sentence by default
- screenshots/images are evidence only when provenance is credible
- generated/editorial images are presentation, not evidence
- if no meaningful visual exists, route back to normal briefing instead of writing text-only visual notes
