# Stop Rules

Stop source gathering when the useful evidence is enough, not when a count is reached.

## By lane

- breaking: major claims have primary/current support or uncertainty is explicit
- topic: recent relevant items are covered and repeats add no new claims
- tech/product/github: official/source pages plus adoption or caveat signal are enough
- research: key papers/clusters and caveats are identified
- market: market data is separately verified from commentary
- social: signal is captured and factual claims are routed for verification
- watchlist: new/material changes since last known state are covered

If the next source would only repeat the same claim, stop and brief.
